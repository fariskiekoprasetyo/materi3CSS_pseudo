Progam Pengembangan Perangkat Lunak dan Gim atau sebelumnya Keahlian Rekayasa perangkat lunak (RPL) adalah salah satu kompetensi keahlian dalam bidang teknologi komputer dan informatika yang secara khusus mempelajari tentang pemograman komputer. Lulusan RPL dewasa ini sangat dibutuhkan untuk menjawab banyaknya kebutuhan industri bagi keteserdiaan tenaga teknisi dalam bidang pengembangan software. Beberapa pekerjaan di bidang RPL adalah software Developer, Game Development, Software Tester, Software Engineering, Software Analist, dan Integrator, konsultan IT dan Progammer

Peluang karir jurusan RPL bisa melingkupi, software engineer/progammer, mobile computing Developer, IT Consultan, Sistem Analist,Game developer, software tester dan masih banyak lagi peluang karir lainnya
